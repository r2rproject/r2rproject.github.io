Reads input from UART.

Combine this function with a char buffer to process inputs from UART

Example:

float kp, ki, kd;
char buffer[max_length];
UART0read(buffer,max_length);
sscanf(buffer, "%f %f %f", kp, ki, kd); // get P, I, D values from user
